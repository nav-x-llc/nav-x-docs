# Catch Weights Setup Page

The **Catch Weights Setup** page is the central configuration location for the entire NAV-X Catch Weights application. This page allows administrators to enable the feature and configure global settings that apply to all items and transactions.

## How to Access

1. Choose the ![Tell me what you want to do](/images/magnifying-glass.gif) icon
2. Enter **Catch Weight Setup**
3. Choose the related link

## Fields

### General Settings

| Field | Description |
| --- | --- |
| **Enable Catch Weight Management** | Toggles the entire Catch Weights feature on or off. When unchecked, no catch weight functionality will be visible in the application. |

### Pricing Configuration

| Field | Description |
| --- | --- |
| **Catch Weight Sales Pricing** | Sets the default pricing method for sales transactions: **Default** - Prices are based on the base unit of measure; **Alternate Unit of Measure** - Prices are based on the catch weight unit |
| **Catch Weight Purchase Pricing** | Sets the default pricing method for purchase transactions: **Default** - Prices are based on the base unit of measure; **Alternate Unit of Measure** - Prices are based on the catch weight unit |

### Tolerance Configuration

| Field | Description |
| --- | --- |
| **Default Alt. Quantity Tolerance %** | The default tolerance percentage for all items. When the alternate quantity variance falls within this tolerance, the outstanding quantity can be automatically updated. Individual items can override this value. |

### Unit of Measure Group Settings

| Field | Description |
| --- | --- |
| **Default UOM Group Blank** | When enabled, newly created item units of measure will not be automatically assigned to a unit of measure group. This allows flexibility in grouping UOMs. |
