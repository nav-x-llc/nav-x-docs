| Field | Description |
| --- | --- |
| **Alternate Unit of Measure** | The unit of measure code that will be used as the catch weight unit for this item. This field is required to enable catch weights for an item. |
| **Catch Weight Sales Pricing** | Determines whether sales prices for this item are based on the base unit of measure (Default) or the alternate catch weight unit of measure (Alternate Unit of Measure). If blank, the global setting from Catch Weight Setup is used. |
| **Catch Weight Purchase Pricing** | Determines whether purchase prices for this item are based on the base unit of measure (Default) or the alternate catch weight unit of measure (Alternate Unit of Measure). If blank, the global setting from Catch Weight Setup is used. |
| **Alt. Quantity Tolerance %** | The tolerance percentage for this specific item. If blank, the default tolerance from Catch Weight Setup is used. Used to handle rounding discrepancies between base and alternate quantities. |
