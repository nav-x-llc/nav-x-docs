# Release Notes for December 2025 Update - Version 1.0.26

## New Features

### Sales Document Import

It is now possible to import multiple lines as complete documents such as Sales Orders and Invoices. **Multi-Line Support** groups integration records into documents based on common identifiers. A new document is automatically created when the grouping field value changes.

### Field Mapping Extensions

Enhanced mapping capabilities have been added for document scenarios, providing more flexible field mapping for multi-line document imports.

### Integration Record Status

Improved tracking has been added for document-level operations, allowing better visibility into the status of document imports.
