# Release Notes for September 2025 Update - Version 1.0.18

## New Features

### Excel Spreadsheet Import

It is now possible to import data directly from **Excel worksheets** (.xlsx files). You can configure whether the first row contains field headers and map Excel columns to Integration Fields by column letter or position.

**Data Preview** allows you to preview Excel data before import mapping. **Multiple Sheet Support** enables processing data from named Excel worksheets.

### File Upload Interface

The file selection and handling interface has been improved for a better user experience when uploading files.

### Data Type Detection

Better handling of mixed data types in Excel columns ensures data is correctly interpreted during import.

### Empty Cell Handling

Proper treatment of empty cells and null values in Excel worksheets prevents errors during import processing.
